SchemeSeeker

A MERN-based web application that empowers Indian citizens to discover government schemes they are eligible for, using smart filtering based on age, gender, income, location, and other criteria.

 Tech Stack

MongoDB – Database

Express.js – Backend Framework

React.js – Frontend Library

Node.js – Server Runtime – Server Runtime



Features

Smart eligibility-based scheme matching – Find schemes that suit you automatically.

Multilingual support – Switch between English and Hindi seamlessly.

User authentication – Sign up, login, and manage your profile.

Favorites – Save your preferred schemes for quick access.

Admin panel – Manage schemes and update information efficiently.

Visual representation – All schemes feature a banner image for clarity and consistency.

Central & State schemes – Access both national and state-level government programs.
